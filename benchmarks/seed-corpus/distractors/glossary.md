# Glossary

Cache: stores a copy of data to serve it faster. TTL: how long a cache entry is
valid. Token: a credential proving identity. Backoff: waiting longer between
retries. Rate limit: a cap on request volume. Failover: switching to a standby.
Feature flag: a runtime switch for code paths.
